# Como instalar?
Baixe o app para o seu respectivo sistema operacional. Em todos eles não é necessário nenhuma dependência adicional para ser executado.

## Windows
Baixe o .exe na versão desejada [aqui](https://github.com/Nasc1mento/xppc-app/releases).

## MacOS
Baixe o .dmg na versão desejada [aqui](https://github.com/Nasc1mento/xppc-app/releases).


## Linux
[![Flathub](https://flathub.org/assets/badges/flathub-badge-en.png)]()

Caso não tenha o repositório do flathub adicionado ou/e não tenha o flatpak instalado, confira [este](https://flathub.org/en/setup) tutorial.
